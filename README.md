# quick-video

A video &amp; audio call app written using golang &amp; react

### YT Video on WebRTC

[WebRTC in 100 Seconds ](https://www.youtube.com/watch?v=WmR9IMUD_CY&ab_channel=Fireship)

### Development

Build images;

```sh
make build-dev
```

For development;

```sh
make run-dev
```

```sh
# List docker images
docker images

# remove docker image
docker rmi <imagename/id>
```
